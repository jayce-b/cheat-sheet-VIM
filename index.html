My Cheat Sheet : VIM pour les debutants

Les Différents Modes dans VIM
par défaut mode variable
i mode insertion
Echap revenir en mode variable
v,V mode visuel
Édition Fichier
:e [nom fichier] ouvre un nouveau fichier (TAB pour l'auto-complétion)
:e/chemin/dossier explorer dans l'arborescence (TAB pour l'auto-complétion)
:w sauvegarder le fichier en cours
:q quitter (msg erreur : impossible de quitter sans sauvegarder
:q! forcer à quitter
:!rm [nomdufichier] supprime le fichier
CTRL-P Trouver des fichiers
v,V mode visuel
Navigation dans le Fichier
k,haut monte une ligne au-dessus
j,bas ligne au-dessous
h,gauche caractère précédent
l,droite caractère suivant
e/E renvoit à la fin mot
gg Se déplacer à la première ligne
GG Se déplacer à la dernière ligne
:[numéro] aller à la ligne numéro
Edition du Texte
~ passer la lettre en maj/min (fonctionne également en mode visuel)
cw modifier le mot
c) modifier jusqu'en fin de phrase
c$ modifier jusqu'en fin de ligne
cc modifier la ligne entière
a/A ajout en fin de ligne
o/O ouvrir une ligne avant la ligne courante/ouvrir une ligne après la ligne courante
r modifier un caractère
R remplace plusieurs caractères
:s/aa/bb/g modification récursive de la ligne (remplace "aa" par "bb"
cs"[mot]" modifie le mot entre guillemet
. répète le dernier changement en mode normal
Suppression
dw supprimer le mot
d) supprimer jusqu'en fin de phrase
D supprimer jusqu'en fin de ligne
dd supprimer la ligne
x supprimer un caractère après le curseur
X supprimer un caractère avant le curseur
